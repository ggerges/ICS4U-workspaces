Javascript Quiz

For this assignment we had to use our knowledge of radio buttons, checkboxes, text fields and java script to code a quiz on a website that would add up the marks from the quiz and display an answer. Unlike everyone else I didn't do a "right" or "wrong" quiz. I did a "which music teacher are you?" quiz.

Questions

  • What was the purpose of the assignment?

    • The purpose of the assignment was to use our new found knowledge of java script, radio buttons, text fields, and check boxes in a interactable quiz.
