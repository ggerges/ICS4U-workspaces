Steps to create my portfolio logo

First, start off with making a new file, name it "Glitter". This will be the background for your text. Make the background transparent and the dimensions 1300 x 870 pixels like as shown in the photo.

Next, go to Layer, New Fill Layer, and then click pattern. This will provid the pattern that we are going to use to make the "Glitter"

When you get into patterns, name your pattern (it doesn't matter what it is named). Then click the button next to the pattern display, press the small arrow next to your loaded patterns, and look for Texture fill 2. This is a default Pattern that Photoshop has. Load the patterns and then pick the very last pattern. Press Ok.

Repeat this step, but this time loading the pattern at 60%. Though you cant see both of them at the same time right now, we will deal with that later.

Next we are gonna get rid of the lines which show that we are using a pattern. However you can not clone on a fill layer so you need to make 2 layers, and place them inbetween the two fill layers. Merge the fill layers with the new layers that you just made. Now you have layers that you can work on.

Click the clone tool on the tool bar and make sure that the hardness is at 100%. The size of the brush is up to you. Select a part of the pattern holding alt and clicking any part of the pattern. The clone tool makes it so that you can "clone" the selection you selected. Draw all over the pattern until you are satisfied with the look. Do this step for the other layer as well.

On the top layer, turn the mode to "Multiply". This will make the patterns both visible.

Make a new layer, and press ctrl alt to select te whole layer. Now, go to edit, located at the top of photoshop, then click fill and fill the layer with the color gold which has the hex id of #ffb700. Press ok.

Now that you have a layer that is filled with the colour gold, turn the blend mode of the layer to "soft light". This will give your pattern the gold effect and makes it look like gold glitter.

Your glitter backgorund is now down! However, now its time to add the text. To do this first we have to save this as a jpg.

Open the newly made glitter background jpg by clicking file,open, then go to layer and duplicate the glitter layer.

Make a new layer, and press ctrl alt to select the full layer. When you do this go to edit, fill, and fill the layer with black. Place the layer inbetween the 2 glitter layers.

Click the top glitter layer and add your text here. You can add as many text layers as you want. In my case I added two. Now, hold ctrl and click each of your text layers and convert them to shapes. You are now going to merge all of the text layers.

Move the Text layer above the black layer but below the top glitter layer. Now click the glitter layer and make it a clipping mask.

Voila! You have your glittery text. But wait, we aren't done yet. Now we have to define the text.

To define the text we are going to add a stroke to it.

Set the stroke settings to this.

Huzzah! You now have a glittered logo! Congratulations!
